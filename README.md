#### 安装教程
需要node/mongodb环境，数据库备份在db文件夹，配置完mongodb后执行

```
// 导入数据库
mongorestore -d wecat ./db
// 跑起来
npm i
npm run serve

// 服务器
cd ./server
npm i
node app
```
